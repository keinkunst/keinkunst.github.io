Grunnatriðin - GitHub og hýsing síðunnar

Við hýsum síðurnar okkar hjá GitHub.

Við ætlum að hýsa síðuna okkar á GitHub. Fyrir því eru nokkrar ástæður:

  • Ókeypis
  • Einfalt að uppfæra síðuna, beint í gegnum VS Code.
  • Ef eitthvað fer úrskeiðis, skjal týnist eða eitthvað álíka, þá er alltaf hægt að sækja gamlar útgáfur á GitHub, ekkert glatast.
  • GitHub er notað svo til allstaðar þar sem margir vinna saman að því að skrifa kóða, svo það er óneytanlega gagnlegt að kynnast því.

Verkefni dagsins fjallar ekki mikið um innihald síðunnar, við ætlum aðallega að koma henni á netið og æfa ferlið sem við munum nota til þess að uppfæra síðuna okkar.

Fylgið eftirfarandi skrefum:

  1. Búðu til Github reikning.
    1. Farðu á síðuna https://github.com/
    2. Búðu til reikning.
    3. Mundu að staðfesta tölvupóstfang og allt slíkt sem gæti verið beðið um.
  2. Gerðu möppuna klára.
    1. Skoðaðu möppuna þar sem þú geymir vefsíðuna þína á tölvunni.
    2. Það er mjög mikilægt að mappan hafi rétt nafn. nafnið skal vera "notendanafn".github.io þar sem notendanafn er þitt notendanafn. T.d. er mitt notendanafn á Github keinkunst, því þarf mín mappa að heita keinkunst.github.io.
  3. Sæktu git fyrir VS Code
    1. Opnaðu VS Code.
    2. Lengst til vinstri á skjánum eru fimm tákn. Þessi valmynd heitir víst Activity bar (ef þú þarft að leita).
    3. Smelltu á miðtáknið til að opna Source Control.
    4. Nú sérðu bláan takka sem segir þér að sækja og setja upp git. Smelltu á takkan, sæktu nýjustu útgáfuna af git og settu hana upp á tölvunni þinni.
    5. Í uppsetningunni þarftu að velja á milli alls konar stillinga. Ekki hafa áhyggjur, þú ættir að geta ýtt á Next þar til git er upp sett, Grunnstillingarnar duga okkur hér.
    6. Eftir að setja upp git, skaltu velja Reaload þar sem boðið er upp á það í VS Code. Ef ekkert breytist við það gætir þú þurft að loka forritinu og opna það aftur.
  4. Settu upp repository
    1. Opnaðu síðuna þína í VS Code og farðu í source control.
    2. Veldu Publish to Github. Þá býrðu til svokallað Repostiory sem verður síðan þín. Gættu þess að velja að hafa það Public og að nafnið sé nákvæmlega rétt, þ.a. "notendanafnið þitt".github.io
    3. Nú ætti vefsíðan þín að vera komin á netið. Wefslóðin þín er þá www."notendanafnið þitt".github.io
  5. Uppfærðu síðuna eftir breytingar.
    1. Nú er allt tilbúið og það verður lítið mál að uppfæra síðuna eftir breytingar.
    2. Láttu fyrstu bretinguna vera að breyta textanum á síðunni á viðeigandi hátt, settu nafnið þitt inn til dæmis.
    3. Til að setja nýj útgáfuna á netið ferðu í Source control.
  6. Opnaðu vefsíðuna þína í VS Code.
    • Fylgið leiðbeiningunum nákvæmlega. Repostitory-ið sem þið búið til þarf að hafa rétt nafn, sem inniheldur notandanafnið ykkar.
    • Veljið góðan stað á drifinu til þess að geyma síðuna.
    • Í ferlinu þurfið þið að setja upp GitHub Desktop. (Ef þið treystið ykkur til þess að nota command promt í staðin þá þurfið þið ekki að sækja það)
    • Afritið öll skjölin sem eiga að vera á síðunni í upphafi og setjið nú í nýju möppuna sem er skráð á GitHub. Þessa upphafssíðu má sækja hér.
    • Athugið að til þess að commit-a breytingar, þarf í GitHub alltaf að skrifa skýringu með. Þá er gott að skrifa eitthvað skiljanlegt eins og “Bætti við lausn á verkefni 4”. En oft er bara verið að uppfæra smáhluti. Textinn skiptir ekki máli, þetta er bara til að hjálpa til við að gefa yfirsýn yfir þróun síðunnar.
  7. Nú getið þið skoðað síðuna á netinu.
  8. Prófið að gera breytingar á kóðanum. Svo til þess að uppfæra síðuna er nóg að nota github í Atom.
    • Í Atom, veljið: Packages -> GitHub -> Toggle Git Tab
    • Allar breytingar sem þið hafið gert frá síðasta commit-i koma undir Unstaged Changes. Smellið á Stage All (uppi til hægri) til þess að færa þær í Staged Changes.
    • Nú geturðu commit-að. Skrifaðu viðeigandi lýsingu á breytingunum.
    • Að commita er bara að staðfesta breytinguna, en hún er ekki enn komin á netið. Smellið á Push, (niðri til hægri) og þá ætti síðan að uppfærast.
  9. Það gæti verið að þú sjáir ekki breytingar á síðunni þinni þegar þú skoðar hana í vafranum. Það gæti verið vegna þess að hann sækir hana ekki á netið, heldur t.d hefur gömlu útgáfuna vistaða í Cookies og sýnir bara hana. Þú getur sótt uppfærðu síðuna með því að gera það sem kallað er Hard refresh. Ætti að vera nóg að ýta á CTRL + F5.
  10. Sendu mér hlekk á síðuna þína í gegnum Innu.
